# 📊 Swiggy Sales & Operational Dashboard

## 🔹 Overview
This project presents an interactive Power BI dashboard built to analyze Swiggy food delivery performance across multiple dimensions such as sales, customer behavior, and delivery efficiency.

## 🔹 Objectives
- Analyze overall sales performance
- Understand customer ordering behavior
- Evaluate delivery efficiency
- Identify key business insights for decision making

## 🔹 Key Metrics
- 💰 **Total Revenue:** 305K+  
- 📦 **Average Order Value:** ₹76.40  
- ⏱️ **Average Delivery Time:** ~40 minutes  
- 🚚 **Delivery Success Rate:** 32%  

## 🔹 Dashboard Features
- 📌 Top cuisines driving revenue  
- 📍 Location-wise sales performance  
- ⏰ Peak order hours analysis  
- 💳 High-value order tracking  
- 🚚 Delivery time vs distance impact  
- 👥 Customer behavior (Online vs Offline trends)  
- ⭐ Rating vs spending insights  

## 🔹 Tools & Technologies
- Power BI  
- Power Query  
- DAX   

## 🔹 Business Insights
- Peak orders occur during specific hours indicating demand spikes  
- Certain cuisines contribute significantly to revenue  
- Delivery time impacts customer satisfaction and ratings  
- High-value customers show different ordering patterns
